# eye-game (gra w oczko)

## 📂 About project
- This project is a ```classic polish card game - oczko``` very similar to blackjack created using ```java```, ```java.awt``` package and also ```mongoDB``` for saving game results
- Project is available only in ```polish language```

## 🎮 Game mechanics:  
- Game has mechanics like blackjack. You can take card or give turn to the dealer pearson who has the nearest point count to ```21```, wins. Also you can take ```2 aces and wins immediately```
- Rest of the rules is in the game main menu under by ```zasady``` button

## ⚙️ Run project
- Project created using ```Intelij IDEA```
- Clone this repository using:
```bash
git clone https://github.com/kkamildev/eye-game.git
```
-   and open using this ```IDEA```
- Idea will prepare project to execution
- You also need to install ```mongoDB```
- MongoDB connection string:
```
mongodb://localhost:27017
```
